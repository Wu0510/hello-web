JS 是什麼

JavaScript：瀏覽器內建的程式語言，負責網頁的行為與互動

HTML  結構     <button>隨機</button>
CSS   外觀     button { background: brown }
JS    行為     按下去 → 隨機選一個、改畫面
  • 和 HTML / CSS 不同，JS 是真的程式語言：有變數、判斷、迴圈、函式
  • 寫在 <script> 裡，或用 <script src="app.js"> 外接
📜 從哪來：1995 年 Netscape 的 Brendan Eich 十天趕工寫出來； 取名 JavaScript 是蹭當年 Java 的熱度，兩者其實無關； 2015 年 ES6 大改版是分水嶺——今天教的 let / const、樣板字串都是 ES6 之後的現代寫法， 看到舊教學滿滿 var 就知道年代了
🔗 參考資料： MDN JavaScript 動態腳本

第一次寫 JS：Console 當計算機

打開 DevTools 的 Console 分頁，直接打 JS、按 Enter 就跑

1 + 1
"好豆" + "咖啡"       // 字串相接 → "好豆咖啡"
console.log("哈囉")   // 印出訊息，寫程式最常用的除錯法
Math.random()        // 隨機一個 0~1 的數
  • Console 是你的即時實驗場：想試什麼語法，這裡打了馬上看結果
  • console.log(...) 是最基本的除錯工具——把值印出來看
🔗 參考資料： MDN Console API

變數：把值存起來

let count = 0;        // 可以之後改的變數
count = count + 1;    // 現在是 1

const name = "好豆";   // 不會再改的，用 const
  • let 宣告「之後會變」的變數；const 宣告「不再改」的常數
  • （舊語法 var 現在少用，先記 let / const）
🐍 對照 Python：Python 直接 count = 0；JS 要先用 let / const 宣告，句尾習慣加 ;

型別：值有哪些種類

let n = 42;            // number   （JS 不分整數/小數）
let s = "拿鐵";         // string
let ok = true;         // boolean  true / false
let empty = null;      // 空值
let arr = ["A", "B"];  // 陣列 array
let obj = { price: 120 }; // 物件 object

typeof n   // "number"
⚠️ 浮點地雷：number 一律是浮點數，小數運算有誤差——Console 敲一次：
0.1 + 0.2 → 0.30000000000000004，所以 0.1 + 0.2 === 0.3 是 false！
這不是 JS 獨有（Python 也一樣），處理金額常用「以分為單位存整數」避開
🐍 對照 Python：number↔int/float、string↔str、boolean↔bool（True/False）、null↔None、array↔list、object↔dict
🔗 MDN 變數、 MDN 型別

運算、比較、樣板字串

10 / 3            // 3.333...
"價格：" + 120     // 用 + 接字串

const price = 120;
`一杯 ${price} 元`  // 樣板字串，用反引號 + ${}

3 === 3           // true（嚴格相等，用三個 =）
3 === "3"         // false（型別不同）
  • 樣板字串（反引號 `）可以把變數用 ${ } 塞進字串，超好用
  • 比較一律用 ===（嚴格相等），別用兩個 ==

truthy / falsy：if 眼中的真與假

if ( ) 裡不一定要放 boolean——任何值都會被當成真或假來看

// 只有這 6 個是 falsy（被當成假），其他「全部」truthy
false   0   ""    null   undefined   NaN
//     零  空字串  空值   還沒給值    壞掉的數字

if ("拿鐵") console.log("會印");    // 有內容的字串 → 真
if ("")    console.log("不會印");   // 空字串 → 假
if (0)     console.log("不會印");   // 0 → 假
  • 背下六個 falsy就好，其他通通是 truthy
  • 超常用：if (name) 一行同時擋掉「沒填」與「空字串」
⚠️ 陷阱：空陣列 [] 和空物件 {} 是 truthy！ 🐍 對照 Python：Python 的空 list / dict 當假，JS 剛好相反
🔗 參考資料： MDN Truthy、 MDN Falsy

判斷與函式

if (price > 100) {
  console.log("小貴");
} else {
  console.log("可以");
}

function greet(name) {
  return `歡迎 ${name}`;
}
greet("好豆");   // "歡迎 好豆"
🐍 對照 Python：Python 用縮排分段落；JS 用 大括號 { }，條件要包在 ( ) 裡

裝一組資料：陣列與物件

const menu = ["美式", "拿鐵", "卡布", "摩卡"];
menu[0]        // "美式"（從 0 開始數）
menu.length    // 4

const drink = { name: "拿鐵", price: 120, hot: true };
drink.name     // "拿鐵"
  • 陣列 [ ]：一串有順序的值，用索引取（第一個是 0）
  • 物件 { }：一組「鍵: 值」，用名字取
🧩 等一下的「中餐轉盤」進階題題，就會把選項存進一個陣列

DOM：JS 操作網頁的橋樑

瀏覽器把你的 HTML，變成一棵可以用程式操作的物件樹（DOM）

document                 整份文件
 └─ html
     └─ body
         ├─ h1
         └─ button        ← 每個標籤都是一個可操作的物件
  • 呼應第 2 堂：Elements 面板看到的那棵樹，就是 DOM
  • JS 透過 DOM 讀取、修改畫面上的元素——這是網頁互動的核心
🔗 參考資料： MDN DOM

抓元素：querySelector

document.querySelector("h1")        // 第一個 h1
document.querySelector(".coin")     // 第一個 class="coin"
document.querySelector("#btn")      // id="btn"
document.querySelectorAll(".coin")  // 全部符合的（一個清單）
  • 用你已經會的 CSS 選擇器語法來抓元素，是一樣的東西
  • 抓到後存進變數，接下來就能改它、對它綁事件
🔗 參考資料： MDN querySelector

改畫面：textContent、classList

const title = document.querySelector("h1");
title.textContent = "換掉的標題";   // 改文字

const coin = document.querySelector(".coin");
coin.classList.add("active");       // 加一個 class
coin.classList.remove("active");    // 移除
coin.classList.toggle("active");    // 有就拿掉、沒有就加上
  • textContent 改文字內容
  • classList 增減 class——搭配 CSS，就能用 JS 切換樣式
🆚 相似三兄弟怎麼選：
textContent——元素裡的全部文字（含被 CSS 藏住的），讀寫都快，預設用它
innerText——只給「眼睛看得到」的文字，會受 CSS 影響、比較慢
innerHTML——會把字串當 HTML 解析；塞使用者輸入會有 XSS 風險，這階段先不用
🔗 參考資料： MDN classList、 MDN textContent（含三者比較）

聽事件：addEventListener

const btn = document.querySelector("#btn");

btn.addEventListener("click", function () {
  console.log("被按了！");
});
  • 事件＝使用者做的動作：click 點擊、input 打字、submit 送出
  • addEventListener("click", 函式)：當這個元素被點，就執行那個函式
🔗 參考資料： MDN addEventListener

隨機：Math.random

Math.random()              // 0（含）~ 1（不含）的小數

Math.random() < 0.5        // 一半機率 true → 拿來擲硬幣

const menu = ["便當", "漢堡", "拉麵"];
const i = Math.floor(Math.random() * menu.length);
menu[i]                    // 隨機選一個品項
  • Math.random() < 0.5：五五波，正好拿來決定正面 / 反面
  • Math.floor(random * 長度)：從陣列隨機挑一個的萬用公式
🔗 參考資料： MDN Math.random

串起來：一個最小互動

<button id="btn">點我</button>
<p id="out">還沒點</p>

<script>
  const btn = document.querySelector("#btn");
  const out = document.querySelector("#out");
  btn.addEventListener("click", function () {
    out.textContent = "你點了！" + Math.random().toFixed(2);
  });
</script>

抓元素 → 綁事件 → 事件裡改畫面：幾乎所有互動都是這個三步驟

小練習 A · 熱身

按一下，正反面切換

  • ⬜️一個按鈕 + 一個 <p> 顯示「正面」
  • ⬜️addEventListener("click", …) 綁到按鈕

主成品：擲硬幣

需求：兩個方塊「正面 / 反面」，按「隨機」→ 隨機高亮其中一個

<!-- HTML -->
<div class="coins">
  <div class="coin" id="head">正面</div>
  <div class="coin" id="tail">反面</div>
</div>
<button id="flip">隨機</button>

/* CSS：高亮用一個 class */
.coin.active { background: gold; font-weight: bold; }

拆成三塊：HTML 結構、CSS 的高亮樣式、JS 的隨機邏輯

擲硬幣：JS 邏輯

const head = document.querySelector("#head");
const tail = document.querySelector("#tail");
const flip = document.querySelector("#flip");

flip.addEventListener("click", function () {
  head.classList.remove("active");   // 先清掉舊的高亮
  tail.classList.remove("active");
  if (Math.random() < 0.5) {
    head.classList.add("active");    // 一半機率正面
  } else {
    tail.classList.add("active");    // 一半機率反面
  }
});

全部都是這堂學過的：querySelector、addEventListener、classList、Math.random
